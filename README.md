# TundraForum
Мное велась вся основная разработка FULL-STACK проекта, по сути сервис для демонстрации и работы на всероссийском шашечном форуме.
Сервис для шашечной партии попеременно с оператором и лучшим СНГ ботом TundraCheckers (для угадывания какой ход  сделал человек, а какой - бот).
Велась основная работа на фреймворке js: Vue; с использовнием mongodb и С++;
Front тоже написан мной. 
Заказ от официального создателя TundraCheckers.
